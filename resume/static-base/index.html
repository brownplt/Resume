about

Resume is a free Web application for faculty applicant submission and review.

Resume is built entirely upon an Ajax-based architecture, thereby combining interactivity with the benefits of remote storage. We designed it to be easy to setup and to have the best interface of any faculty review software. See for yourself with our guided tour.

Resume is available completely free of cost. For more information, please contact Arjun Guha at arjun@cs.brown.edu.
